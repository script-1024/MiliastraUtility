# 千星沙箱工具集 (Miliastra Utility)
用于千星奇域的非官方实用程序，目标是以程序方式实现对 GIx 文件自动化、批量化、可定制化的创建和修改。

想了解千星奇域的文件格式可参考隔壁项目的文档：🔗[链接](https://github.com/script-1024/genshin-miliastra-file-format)

## 事前提醒

1. 所有通过程序生成的文件最终还是要被导入回游戏中，你**无法绕过任何游戏自身的限制**，该接受的上传审核一样不会少，因此请务必遵守原神的[用户协议](https://ys.mihoyo.com/main/company/agreement)

2. 本项目仍处于**早期阶段**，远未达到可用于生产环境的程度，**暂时不提供可执行文件**！但你可以将项目下载到本地后自行编译一份来玩玩看 😉
